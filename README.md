# VCF Converter

React застосунок, створений за допомогою Vite.

## Встановлення

```bash
npm install
```

## Запуск

Для запуску сервера розробки:

```bash
npm run dev
```

Відкрийте [http://localhost:5173](http://localhost:5173) у браузері.

## Збірка

Для створення production збірки:

```bash
npm run build
```

Для перегляду збірки:

```bash
npm run preview
```
